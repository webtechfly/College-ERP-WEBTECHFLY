# 🎓 College ERP System -- WEBTECHFLY

**A Modern MERN Stack Based Academic Management Platform**

The **College ERP System** developed by **WEBTECHFLY** is a
comprehensive web-based academic management platform designed to
digitalize and automate the administrative and academic operations of
educational institutions.

Built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**,
the system provides a centralized dashboard where **administrators,
faculty members, and students** can manage academic data efficiently.

The platform eliminates traditional paper-based processes and introduces
a **secure, scalable, and user-friendly digital ecosystem** for managing
institutional information such as student records, faculty management,
attendance tracking, subjects, examinations, and academic reports.

------------------------------------------------------------------------

# 🚀 Project Overview

Educational institutions often struggle with managing large volumes of
academic and administrative data manually. The **WEBTECHFLY College ERP
System** simplifies this process by providing a unified platform that:

-   Improves operational efficiency
-   Reduces manual data handling
-   Enhances transparency across departments
-   Provides real-time academic data access

# ⚙️ System Configuration & Setup

## 1️⃣ Clone the Repository

``` bash
git clone https://github.com/webtechfly/College-ERP-WEBTECHFLY.git
cd College-ERP-WEBTECHFLY
```

## 2️⃣ Environment Configuration

Create a `.env` file inside the **server** directory.

Example:

    MONGODB_URI=your_mongodb_connection_string
    JWT_SECRET=your_secret_key
    PORT=5000

## 3️⃣ Install Dependencies

### Frontend

``` bash
cd client
npm install --legacy-peer-deps
```

### Backend

``` bash
cd ../server
npm install
```

## 4️⃣ Start the Application

### Start Backend

``` bash
npm start
```

### Start Frontend

``` bash
cd client
npm start
```
